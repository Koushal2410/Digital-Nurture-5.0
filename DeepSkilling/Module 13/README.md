# Module 13 - Agile Methodology

This module covers Agile software development principles, the Scrum framework, sprint metrics, and writing user stories.

---

## Key Learning Areas

### 1. Agile Principles
- **Agile Manifesto**: 4 core values and 12 principles prioritizing individuals, working software, collaboration, and responding to change.
- **Agile vs Waterfall**: Iterative cycles vs sequential linear development phases.

### 2. Scrum Framework
- **Scrum Roles**: Product Owner, Scrum Master, Developers.
- **Scrum Ceremonies**: Sprint Planning, Daily Scrum, Sprint Review, Sprint Retrospective.
- **Scrum Artifacts**: Product Backlog, Sprint Backlog, Increment, Definition of Done (DoD).

### 3. User Stories & Estimation
- **INVEST Principle**: Independent, Negotiable, Valuable, Estimable, Small, Testable.
- **User Story Format**: As a [user], I want [goal], so that [benefit].
- **Agile Estimation**: Story points, Planning Poker, velocity, and burndown charts.
