# Module 14 - Gen AI Fundamentals

This module covers Generative Artificial Intelligence concepts, Prompt Engineering techniques, and using AI coding assistants like GitHub Copilot.

---

## Key Learning Areas

### 1. Generative AI Basics
- **Generative AI Definition**: AI models capable of generating new content (text, code, images, audio) based on training data.
- **Generative vs Discriminative**: Creating new patterns vs classification/prediction.
- **Evolution**: Transformers, Large Language Models (LLMs), GPT, and Codex.

### 2. Prompt Engineering Techniques
- **Zero-Shot Prompting**: Requesting tasks without giving examples.
- **Few-Shot Prompting**: Providing a few examples of input/output to guide format.
- **Chain-of-Thought (CoT)**: Instructing models to reason step-by-step to arrive at conclusions.
- **Best Practices**: Being specific, providing clear constraints, iterating, and specifying output formats.

### 3. GitHub Copilot
- **AI Pair Programming**: Auto-completing lines, generating functions from comments, writing documentation, and generating unit tests.
- **Ethics & Risks**: Identifying hallucinations, license compliance (copyleft), security vulnerabilities, and privacy policies.
